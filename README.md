## Getting Started

### Prerequisites

```bash
node >= 18.0.0
npm >= 9.0.0
```

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/gibberlink.git
```

2. Install dependencies
```bash
cd gibberlink
npm install
```

3. Set up environment variables
```bash
cp .env.example .env
```

### Running the Application

```bash
npm run dev
```
